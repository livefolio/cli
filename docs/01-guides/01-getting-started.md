# Getting started with Livefolio CLI

`@livefolio/cli` is a command-line interface for accessing Livefolio market data and strategy evaluation, built on `@livefolio/sdk`. It is designed to be used by both humans and AI agents — all output is structured as CSV or JSON, and every command supports standard shell piping and redirection.

This documentation serves as both a specification of CLI functionality and a practical usage guide. The **Guides** section covers installation, authentication, and command conventions. The **Modules** section provides detailed documentation for each command group: auth, market, portfolio, and strategy.

## Installation

Requires Node.js 18 or later.

```shell
npm install -g @livefolio/cli
```

Verify the installation:

```shell
livefolio --version
```
